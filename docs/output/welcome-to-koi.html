Koi Documentation

Welcome to Koi

Koi is a little scripting language similar to JavaScript

Features

  • Classes
  • Functions
  • Standard Library

Code Sample

class Koi {
  init(name) {
    this.name = name;
  }
  swim(to) {
    print("Swimming to " + to);
  }
}
var koi = Koi("fishy");
koi.swim();
  • Koi is multi-paradigm
  • Koi is small: there are only 911 semicolons in the entire codebase
